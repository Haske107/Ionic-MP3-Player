# Ionic-MP3-Player
Ionic MP3 Player

# Getting Started

Navigate to directory:
`cd Ionic-MP3-Player`

NPM:
`npm install`

Ionic Framework:
https://ionicframework.com/getting-started

Have Ionic installed globally:
`npm install -g ionic`

Run application:
`ionic serve`
